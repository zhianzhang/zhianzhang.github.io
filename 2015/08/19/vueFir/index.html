vue的构造器

  • vue 的实例里面存放了页面显示的数据，属性，和方法
    这个实例通常叫做Vue的根实例

  • 组件构造

    • vue 最强大的功能是生成可重复使用的组件

注意

  • 我们要注意的是 vue 的数据和 Angular 的数据一样都会产生联动的效果
    实例 vm.name = "kkk"; 这样的赋值同样会引起视图中的 name 发生改变

  • 同样 data.nae = "aaa";它会引起视图和实例的变化

  • 如果在实例创建之后添加新的属性到实例上，它不会触发视图更新。这与 vue 的响应系统有很大的联系

  • 实例的属性和方法

    • 为了区别实例的属性方法和代理数据，我们为每个实例
      方法都添加了$。

    — 我们检查 vm.$data === data,发现结果是true
    — 我们检查 vm.$el === document.getElementById("app")
    — 同样 vue 官方为我们提供了实例的众多方法如：vm.$watch()、
    vm.$get()、 vm.$set()等
