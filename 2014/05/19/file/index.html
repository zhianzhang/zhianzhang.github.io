有关 CSS （层叠样式表：Cascading Style Sheets）

CSS用来表现HTML（标准通用标记语言的一个应用）
或XML（标准通用标记语言的一个子集）等文件样式的计算机语言。（不是编程语言！）
能够真正做到网页表现与内容分离的一种样式设计语言，实现HTML结构和样式的精确分离
提高后期运维人员的可维护性和易维护性。这样的开发理念是众多开发者的福音
同时作为计算机领域的国际性组织 W3C 的作用也是
功不可没的。我们要注意的是CSS样式不能单独存在，它必须依赖HTML的元素（ Tag/Element）

CSS的历程

  • CSS1

    在网页发展的历程中，样式的存在吸引了更多的用户来访问某个网页，早在浏览器大战
    时代。开发者更多的是利用标签来展示样式

    当网页的内容越来越多，样式越来越复杂的时候。开发者发现在页面的结构中
    没必要的东西越来越多。开发者急需一种新的技术来解决这种日益突出的矛盾。
    作为一项 W3C 推荐，CSS1 发布于 1996年12月17 日。1999 年1月11日，此推荐被重新修订，
    此后终于解决了样式行为的标签所带来的杂乱开发的无奈。

  • CSS2

    在CSS1的基础上添加了对媒介（打印机和听觉设备）和可下载字体的支持，
    我们现在所说的 css 就是 css2，这个版本的发布时间是1999年1月11日。
    这个时代的媒体和设备不如现在这么发展迅速。所以这个版本是所有浏览器
    支持最多的（IE有部分自己的样式书写方式）

CSS3的状况

现代浏览器已经快速更近了 CSS3 的脚步，绝大部分属性得到了支持。这是因为
CSS3 的发布不是一次性发布的。它分模块发布，这样给大部分浏览器厂商有了提高
自己产品的充分开发时间
